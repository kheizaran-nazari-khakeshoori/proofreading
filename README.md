# Special Texts Website

A small static website that shows five special text cards.

## Run locally

From the project folder:

```bash
cd /home/kheizaran/Github/proofreading
python3 -m http.server 8000
```

Then open:

```
http://localhost:8000
```

## Files

- `index.html` — page structure
- `styles.css` — styling for the card layout
- `main.js` — renders the five special texts and updates the year
- `.gitignore` — ignore file rules
